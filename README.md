# Khaled Coffee

## Description
Khaled Coffee est une application de gestion de café offrant une expérience utilisateur optimale pour les amateurs de café.

## Fonctionnalités
- Gestion des commandes
- Catalogue de produits
- Système de paiement
- Historique des achats
- Interface utilisateur intuitive

## Installation
```bash
git clone https://github.com/mehanab/KhaledCoffee.git
cd khaledCoffee
npm install
```

## Utilisation
```bash
ionic serve

ionic build
```

## Technologies
- Node.js
- ionic/vue 8.0.0
- vue 3.3.0
